# AI-Based Real-Time Object Detection System

## 📌 Project Overview

This project implements a real time object detection system using computer vision and deep learning techniques. The system processes live video input from a webcam and detects multiple objects in real time by generating bounding boxes and class labels.

The primary objective of this project is to understand the design and implementation of AI-based vision systems, focusing on performance, accuracy, and real time processing.

---

## 🧠 Technical Implementation

### 🔹 Model & Approach

* Object Detection Algorithm: YOLO v5
* Detection Type: Real-time multi-object detection
* Pre-trained models used for inference

### 🔹 Tools & Frameworks

* Programming Language: Python
* Libraries: OpenCV, NumPy
* Deep Learning Framework: TensorFlow, PyTorch 

### 🔹 System Workflow

1. Capture video stream from webcam
2. Convert frames into processable format
3. Apply object detection model on each frame
4. Generate bounding boxes with labels
5. Display output in real time
